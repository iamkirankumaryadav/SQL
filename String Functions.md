# `String` Functions and Operations

### `CONCAT`

```sql
SELECT CONCAT(FirstName, ' ', LastName) AS Name;
```

Name
:---
Kirankumar Yadav

### `UPPER`

```sql
SELECT UPPER('Kirankumar Yadav') AS Name;
```

Name
:---
KIRANKUMAR YADAV


### `LOWER`

```sql
SELECT LOWER('Kirankumar Yadav') AS Name;
```

Name
:---
kirankumar yadav

`TRIM`

```sql
SELECT "  Kirankumar Yadav  " AS Name;
```

Name
:---
Kirankumar Yadav
